## âœï¸ Customization Guide

### Change Colors
Edit CSS variables in `style.css`:
```css
:root {
    --lavender: #b19cd9;        /* Your primary color */
    --dark-lavender: #9370db;   /* Your secondary color */
    --light-lavender: #e6e0f0;  /* Your light color */
}
```


### Add More Projects
Copy and paste a project card in `index.html` and update the content.

### Modify Skills
Edit the skills section in `index.html` and update the `data-progress` attribute (0-100).
